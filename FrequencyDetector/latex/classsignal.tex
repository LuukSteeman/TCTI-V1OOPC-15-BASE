\subsection{Detailed Description}
Signal class for calculating frequency, intensity and average. 

This is the class used to store the signal data received from the Aruino Due A\+DC and calculate the various elements of a wave. This class was made with soundwaves in mind and might therfore not work with other types of waves. All characteristics of this class are private and are only made visible by the class\textquotesingle{}s functions. All function definitions can be found in the signal\+\_\+pin.\+cpp file.

Calculate the wave intensity. 

This function calculates the wave\textquotesingle{}s intensity. It does so by calculating the average deviation from the wave\textquotesingle{}s average, by comparing all the values to the return value of \hyperlink{classsignal_a835580fc0e7fbb50caca191ba16a9f44}{find\+\_\+average()}. The returned value is abstract and does not directly corrolate to DB. Higher returned values does however corrolate to a higher sound level.
