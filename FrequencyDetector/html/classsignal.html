Detailed Description

Signal class for calculating frequency, intensity and average.

This is the class used to store the signal data received from the Aruino Due ADC and calculate the various elements of a wave. This class was made with soundwaves in mind and might therfore not work with other types of waves. All characteristics of this class are private and are only made visible by the class's functions. All function definitions can be found in the signal_pin.cpp file.

int signal::find_frequency ( )

Calculate the wave frequency.

This function calculates the frequency of the wave in Hz and returns it. It does so by first calling the fin_average() function, and checking how often it passed the value found by the function. The resulting "crossover" value is then multiplied by a calibration factor. If you find the frequency to be off, use a known frequency and change the factor until the function returns the desired frequency.

int signal::find_intensity ( )

Calculate the wave intensity.

This function calculates the wave's intensity. It does so by calculating the average deviation from the wave's average, by comparing all the values to the return value of find_average(). The returned value is abstract and does not directly corrolate to DB. Higher returned values does however corrolate to a higher sound level.
